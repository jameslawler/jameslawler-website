Ghost is an easy to use CMS platform for building easy, but also feature full web sites. It is mainly targeted at people who want to have paying customers with lots of membership features. The platform itself is open source and you can run your own instances of Ghost very easily using Docker. If you want a fully managed environment you an pay for their hosted version of Ghost.

This post is about trying to achieve the following goals with Ghost;

  • static version (without membership features)
  • running for free on GitHub pages
  • automated build and deployment using GitHub Actions
  • version control of changes
